### CoreUHC
A Unique UHC plugin!
## Features
- Teams

- Scenarios(soon)

- Customizable team sizes


## Installation
1) Drop the plugin(folder format or .phar) into /root/plugins/ 

2) Start the server

3) Customize the config.yml to your needs

4) Stop/start the server

5) You are ready to start a UHC match!
